Yelp Data Challenge

Designed and developed a stand-alone Java application which runs queries on the Yelp data. Created DML and DDL statements for Oracle database to insert JSON data from Yelp dataset, supporting it with a Java Swing UI. The dataset includes 20,544 businesses, 252,898 users, and 1,125,458 reviews gathered from Phoenix, Las Vegas, and Madison.

Fortified Cards

Google Map Nav Add-On

Co-designed and developed an Android application, using Google Maps API, to extend current Google Maps, such that an optimal route between source and destination via a stop – off location is displayed.

3D Holographic Projection
